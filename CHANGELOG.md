# Changelog

## [1.0.0-benpay] - 2026-04-24

### Added

- BenPay branding defaults on login and authenticated header views.

- New account creation flow for first-time client onboarding from the login page.
- Repository modernization assets:

  - GitHub Actions CI workflow for PHP linting.
  - Root `LICENSE` file for clearer license visibility.

### Changed

- Login screen structure updated to include auth tabs for login and registration.
- Theme classes wired consistently between login and authenticated pages.
